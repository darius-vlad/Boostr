Boostr is a full-stack crowdfunding platform designed to connect startup founders with early-stage backers. It enables founders to pitch their ideas and allows the community to discover, fund, and rank emerging startups.

**Key Features:**
* **Campaign Management:** Founders can create detailed project pitches with funding goals.
* **Donation System:** Secure payment integration for backing projects.
* **Discovery:** An "Explore" page with filtering to find trending startups.
* **Gamification:** Live Leaderboards tracking top-funded projects and top contributors.
* **Tech Stack:** React, Node.js, Express, SQL Database.
